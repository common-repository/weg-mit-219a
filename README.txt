Dieses Plugin bindet eine Schaltfläche auf deiner Frontpage ein, die deine Besucher über Schwangerschaftsabbrüche informiert.

== Description ==
Am 24.06.2022 wurde der Paragraph endlich aus dem Strafgesetzbuch gestrichen. Die Existenz dieses Plugins darf gern daran erinnern.

Laut §219a StGB ist es verboten über Schwangerschaftsabbrüche zu informieren, wenn man sie anbietet. Dafür ist es aber gestattet, über sie zu informieren, wenn man sie nicht anbietet.

Dieses Plugin ist also für alle, die über Schwangerschaftsabbrüche informieren möchten und können, um einerseits die Absurdität dieser Gesetzgebung heraus zu kehren und andererseits für einen offenen und freien Umgang mit medizinischen Informationen sind.

WEG MIT §219a!

Features:
* freie Positionierung der Schaltfläche in den Ecken des Browser
* Anzeige der Schaltfläche bei Posts, auf Seiten oder beides oder gar nicht
* Einbindung der Schaltfläche über einen Shortcode
* freie Gestaltung der Schrift- und Hintergrundfarbe
* Ausblenden der Schaltfläche für angemeldete User

== Installation ==
1. Import der Datei weg-mit-219a.zip über den Plugin-Upload oder 
das entpackte Verzeichnis \"Weg mit 219a\" in das Verzeichnis \"/wp-content/plugins/\" hochladen
2. Aktivieren des Plugins über den Plugin-Menüpunkt in Wordpress
3. ggf. einbinden des Shortcodes mittels \"do_shortcode( \'[wegmit219a]\')\" an geeigneter Stelle

== Frequently Asked Questions ==
= Darf ich das Plugin verwenden, wenn ich Schwangerschaftsabbrüche anbiete? =
Ja, aber laut §219a StGB wäre das illegal und Du könntest angezeigt werden.

= Darf ich das Plugin verwenden, wenn ich keine Schwangerschaftsabbrüche anbiete? =
Ja.

= Ich arbeite in einer Backstube, Anwaltskanzlei, Social Media Marketing Agentur... =
Ja.

= Warum kann ich es für angemeldete User ausblenden =
Also manche vorgesetzten Personen... Du verstehst.
